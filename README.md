# Description:
Expense-Splitting Application.

# System:
Linux Mint Kernel

# Requirements:
1. Ruby on Rails
2. React
3. PostgreSQL

# Setup:
$ npm install (https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

$ rvm install 2.5.1 (https://rvm.io/rvm/install)

$ rvm use 2.5.1

$ bundle update

$ bundle install

$ rake db:setup

# Run Frontend:
$ npm run webpack

# Run Backend
$ rails s
